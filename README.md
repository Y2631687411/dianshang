# dianshang
我构建了一个跨境电商 AI 运营 Agent，用于自动完成选品分析、竞品研究、用户痛点提炼、英文 Listing 生成和关键词优化。它解决的核心痛点是中小卖家在上架商品时需要花大量时间研究竞品、整理评论痛点、撰写英文标题和五点描述，效率低且文案容易同质化。  该系统采用多 Agent 协作流程：选品 Agent 负责判断商品潜力，竞品 Agent 分析价格、标题和卖点，评论 Agent 总结用户痛点，文案 Agent 生成英文标题、五点描述和详情页内容，关键词 Agent 负责搜索词优化，质检 Agent 检查语法、合规性和平台风格。整个流程包含“市场调研—竞品分析—痛点总结—卖点重构—文案生成—质量检查”的长链推理，最终输出完整的商品上架方案，可帮助运营人员明显减少重复劳动，提高上架效率。
